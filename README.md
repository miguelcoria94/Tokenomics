# Tokenomics

A crypto market app built with Django and tailwind that lets create a watchlist and view crypto market details.

## Features

- Login/Logout
- Each user gets a watchlist
- Market Overview
- Single Coin Details
- Add to watchlist
- Remove from watchlist
- Convert USD to COIN Price
- Coin Charts
